# SIMS – Sistema de Gestión Integral de Movilidad Sostenible

**Proyecto:** Frontend Web Multi-tenant de Gestión de Flota  
**Estado:** Sprint 5 – Producción  

SIMS es una plataforma avanzada diseñada para la gestión de servicios de movilidad sostenible. Este repositorio contiene el frontend desarrollado en **Vue 3**, enfocado en la eficiencia operativa, la escalabilidad multi-inquilino y una experiencia de usuario intuitiva apoyada por Inteligencia Artificial.

## Características Principales

- **Gestión Multi-tenant:** Aislamiento total de datos por organización.
- **Mapas en Tiempo Real:** Visualización y seguimiento de flota mediante Leaflet.
- **Ayuda Inteligente:** Chatbot con arquitectura RAG conectado a la IA del instituto para soporte inmediato.
- **Sistema de Soporte:** Gestión completa de tickets de mantenimiento y atención al cliente.
